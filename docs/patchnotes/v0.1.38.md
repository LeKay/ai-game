# v0.1.38 — First Playable Version

First release of the game skeleton: main menu, save/load, map, tick system, actions.

## New
- Main menu with **New Game**, **Continue** and **Quit** (including fade-in/out and version label)
- Auto-save and Continue via `WorldSaveManager`
- Map scene with terrain rendering, building indicators and path overlay
- Drag-and-drop for resources directly on the map
- Tick system with pause and multiple speed levels
- NPC system including hunger, logistics carriers and tasks
- Crafting registry for production chains
- Inventory and logistics system with routes
- Progression tree and overworld view
