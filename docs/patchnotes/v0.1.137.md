# v0.1.137 — Clay Pit Placement, Perk Rework & Production Speed Editing

## Highlights
The Clay Pit now sits directly on clay terrain (like the Bridge on water). The Calling perk gains a second effect — production speed — and Carriers get their own calling. Production rate labels are now editable fields, letting you dial in a target throughput directly in the building panel.

## New
- **Clay Pit on clay tiles**: The Clay Pit is now placed directly on CLAY terrain, not adjacent to it. Grid placement routes it through the same terrain-occupation path as the Bridge on water.
- **Carrier calling**: Logistics NPCs can now choose the **Carrier** calling at their first level-up — grants +25% delivery XP and +1 carrier capacity. The Carrier card only appears when the NPC has active routes.
- **Editable production rates**: The per-resource rate labels in the building production section (`⚙` settings toggle) are now editable `LineEdit` fields. Type a target rate (items/day) and press Enter to change the building's production speed.
- **Day overview: calling label**: When an NPC levels up from level 1 for the first time, their calling is shown in gold below the XP bar.
- **Item settler system**: `WorldSaveManager` now has a `register_item_settler()` hook. Before any disk save or map snapshot, all registered settlers flush in-flight transport items back into serialized storage so no item is lost across save/load.

## Changed
- **Calling perk rebalanced**: XP bonus reduced from +50% to +25%; a new secondary effect adds +10% production speed for the profession building type while supplied. Both effects trigger together.
- **Perk cards pre-rolled on level-up**: Perk cards are now generated the moment an NPC levels up and stored in the save file. Closing and reopening the perk picker no longer rerolls the options.
- **Calling card priority**: The guaranteed first slot is the NPC's assigned building type (if unlocked), or Carrier (if they have active routes). Remaining slots are shuffled other buildings.
- **Build placement energy check removed**: Energy is no longer checked at tile placement. A construction action is queued on the player instead — energy is consumed during construction, not at the moment of clicking.
- **Plains terrain**: Reduced tree density on plains (~25% → ~17%) by widening the empty/grass elevation band so open plains read less wooded.

## Balance
- **Axe (Lumber Camp with tool)**: Output 5 → 10 wood per cycle.
- **Pickaxe (Stone Mason with tool)**: Output 5 → 11 stone per cycle.
- **Fishing Hut**: Cycle duration 250 → 500 ticks (halved output rate).
- **Spindle (Workbench recipe + crafting bench)**: Output quantity 1 → 2 per craft.

## Fixes
- **Crafting race condition**: The crafting system now reserves output storage at the start of a craft. Transport routes finishing mid-craft can no longer fill the same slot, causing finished items to disappear.
- **Camera key leak**: Camera now clears held pan keys and cancels middle-drag when the input context changes (e.g. opening a drawer), preventing stuck movement after UI interaction.
- **Demolish overlay**: Demolish mode now uses a dedicated `ColorRect` overlay instead of reusing the ghost texture's modulate, preventing ghost-texture tinting on other modes.
- **Save/load construction actions**: Construction (`CONSTRUCT_BUILDING`, `CONSTRUCT_PATH`) actions restore correctly across save/load — they no longer look up a `ManualActionConfig` that doesn't exist for them.
- **Tick pause on load**: `TickSystem` is now explicitly paused after `apply_pending_load()` to prevent a tick burst while the scene settles.
- **Perk secondary effects**: `npc_perk_bonus` / `building_perk_bonus` / `building_has_active_perk` now check both primary and secondary effect fields on each perk definition.
- **Calling type display**: `get_npc_job_name()` correctly shows "Carrier" for NPCs whose saved profession is `PROFESSION_CARRIER`.

## Technical
- `WorldGrid.validate_clay_placement()` / `place_building_on_clay()` — new grid layer for clay-terrain buildings.
- `BuildingRegistry._validate_grid_placement()` / `_place_building_on_grid()` — routing for Clay Pit added alongside Bridge.
- `PerkRegistry.PROFESSION_CARRIER = -2` — sentinel for carrier profession (distinct from -1 "no profession").
- `PerkRegistry.EFFECT_CALLING_BUILDING_EFF` — new effect constant for Calling's production speed bonus.
- `CraftingRegistry._CRAFT_HOLDER_ID` — fixed holder id for storage reservations during crafting.
- `WorldSaveManager.register_item_settler()` / `_settle_in_flight_items()` — pre-save item flush hook.
- `ProgressionSystem._resources_for_unlock()` — `"building"` entries no longer pre-populate reachable resources; only `"building_recipe"` entries do.

## Known Issues
- Changes have not yet been verified headless through the test suite / Godot editor.
