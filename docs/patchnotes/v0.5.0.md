# v0.5.0 — Patchnotes im Hauptmenü

## Neu
- **Patchnotes-Button** im Hauptmenü öffnet ein scrollbares Fenster mit allen Versionshinweisen
- Patchnotes werden zur Laufzeit aus `docs/patchnotes/*.md` geladen — neueste Version oben
- Markdown-Subset (Überschriften, Listen, **bold**, *italic*, `code`) wird zu BBCode konvertiert
- ESC oder X schließt das Fenster

## Technisch
- Neue Szene `src/ui/screens/patchnotes_screen.tscn` (+ Script)
- `export_presets.cfg` packt `docs/patchnotes/*.md` in den Build
