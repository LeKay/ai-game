# v0.1.127 — Level-basierte Perk-Gruppen

NPCs bekommen Perks aus einem level-abhängigen Goods-Pool. Höhere NPC-Level schalten neue Warengruppen frei.

## Neu
- `perk_group`-Feld auf allen Ressourcen
- **LEVEL_PERK_GROUPS**-Mapping inkl. Helper für den level-restringierten Goods-Pool
- **Gruppe 1** (Level 1+): Kleidung
- **Gruppe 2** (Level 3+): Töpferwaren, Leder, Möbel
- NPC-Level-Cap auf 3 festgesetzt
- Tests decken die level-basierte Pool-Restriktion ab

## Verbessert
- Gebundene Goods werden mit Replacement aus dem level-restringierten Pool gezogen
- Das ungenutzte `perk_eligible`-Feld wurde entfernt

## Dokumentation
- Design-Spec und Implementation-Plan für level-basierte Perk-Good-Gruppen
