# v0.1.127 — Level-Based Perk Groups

NPCs gain perks from a level-dependent goods pool. Higher NPC levels unlock new goods groups.

## New
- `perk_group` field on all resources
- **LEVEL_PERK_GROUPS** mapping including a helper for the level-restricted goods pool
- **Group 1** (level 1+): clothing
- **Group 2** (level 3+): pottery, leather, furniture
- NPC level cap set to 3
- Tests cover the level-based pool restriction

## Improved
- Bound goods are drawn with replacement from the level-restricted pool
- The unused `perk_eligible` field was removed

## Documentation
- Design spec and implementation plan for level-based perk-good groups
