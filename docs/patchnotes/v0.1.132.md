# v0.1.132 — Patch Notes in the Main Menu

## New
- **Patch notes button** in the main menu opens a scrollable window with all version notes
- Patch notes are loaded at runtime from `docs/patchnotes/*.md` — newest version on top
- A Markdown subset (headings, lists, **bold**, *italic*, `code`) is converted to BBCode
- ESC or X closes the window

## Technical
- New scene `src/ui/screens/patchnotes_screen.tscn` (+ script)
- `export_presets.cfg` bundles `docs/patchnotes/*.md` into the build
